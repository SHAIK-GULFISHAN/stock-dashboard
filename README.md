# 📈 Stock Market Live Dashboard

## Description
A Django-based Stock Market Dashboard that allows users to search stocks and manage their investment portfolio.

## Features
- User Login
- Portfolio Management
- Add Stocks
- View Stock Holdings
- Django Authentication
- Responsive UI using Bootstrap

## Technologies Used
- Python
- Django
- HTML
- CSS
- Bootstrap
- SQLite

## Installation

```bash
git clone https://github.com/SHAIK-GULFISHAN/stock-dashboard.git
cd stock-dashboard
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

## Project Structure

```
stock-dashboard/
```
